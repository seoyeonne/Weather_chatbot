# Weather_chatbot
<br>

## 날씨를 알려주는 챗봇 서비스

<br>

### 내용


<br>

### 도구

* Dialogflow
* replit

<br>

### 기능

* 지역별 오늘, 내일 날씨 알림
* 날씨별 놀거리 추천 사이트 제공
* 일주일 상세 날씨 사이트 제공
* 월별 공휴일 알림

<br>

### 구현내용

1) Dialogflow에서 여러 인텐드, 엔터티 작성
2) replit에서 날씨를 크롤링하여 목적에 맞는 날씨 반환

<br>
